## Official Web Standards & Documentation

- [MDN Web Docs](https://developer.mozilla.org/) – Authoritative resource for HTML, CSS, JavaScript, and browser APIs.
- [WHATWG HTML Standard](https://html.spec.whatwg.org/) – Living HTML standard maintained by browser vendors.
- [W3C](https://www.w3.org/) – Official web standards body for CSS, accessibility, and web architecture.
- [ECMAScript Language Spec](https://tc39.es/) – Specification for JavaScript maintained by TC39.
- [Can I Use](https://caniuse.com/) – Browser support tables for modern and experimental web technologies.

## Frontend Frameworks

- [React](https://react.dev/) – Component-based UI library for building interactive frontends.
- [Vue.js](https://vuejs.org/) – Progressive framework for building approachable and efficient interfaces.
- [Angular](https://angular.io/) – Full-featured framework for complex enterprise-grade applications.
- [Svelte](https://svelte.dev/) – Compiler-driven UI framework for highly performant interfaces.
- [SolidJS](https://www.solidjs.com/) – Reactive UI library for fine-grained performance.

## CSS Frameworks & UI Libraries

- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework for rapidly styling components.
- [Bootstrap](https://getbootstrap.com/) – Popular CSS framework for responsive and accessible UI elements.
- [Bulma](https://bulma.io/) – Modern CSS framework based on Flexbox.
- [Material UI](https://mui.com/) – React component library implementing Google’s Material Design.
- [Chakra UI](https://chakra-ui.com/) – Accessible React component library with themeable components.

## JavaScript Libraries & Utilities

- [jQuery](https://jquery.com/) – Widely used DOM manipulation and event handling library.
- [Lodash](https://lodash.com/) – Utility library offering functional helpers for arrays, objects, and data.
- [Day.js](https://day.js.org/) – Lightweight alternative to Moment.js for date/time manipulation.
- [Axios](https://axios-http.com/) – Promise-based HTTP client for browsers and Node.js.
- [D3.js](https://d3js.org/) – Data-driven visualization library for interactive charts and graphics.

## Static Site Generators

- [Next.js](https://nextjs.org/) – React framework with SSR, SSG, and serverless API routes.
- [Gatsby](https://www.gatsbyjs.com/) – React-based SSG optimized for performance and content sites.
- [Hugo](https://gohugo.io/) – High-performance static site generator written in Go.
- [Jekyll](https://jekyllrb.com/) – Ruby-based SSG powering GitHub Pages.
- [Astro](https://astro.build/) – Content-focused SSG with zero-JS by default.

## Backend Frameworks

- [Node.js](https://nodejs.org/) – JavaScript runtime for backend, API, and full-stack applications.
- [Express.js](https://expressjs.com/) – Minimal framework for building REST APIs in Node.js.
- [NestJS](https://nestjs.com/) – TypeScript backend framework for scalable server-side applications.
- [Ruby on Rails](https://rubyonrails.org/) – Full-stack MVC framework for rapid web application development.
- [Django](https://www.djangoproject.com/) – Python framework with batteries-included philosophy.
- [Laravel](https://laravel.com/) – Elegant PHP framework with expressive syntax and rich tooling.

## APIs, GraphQL & Server-Side Tools

- [GraphQL](https://graphql.org/) – Query language for APIs enabling strongly typed data fetching.
- [Apollo GraphQL](https://www.apollographql.com/) – Complete GraphQL platform for servers and clients.
- [tRPC](https://trpc.io/) – TypeScript-first framework for creating end-to-end typesafe APIs.
- [Express.js](https://expressjs.com/) – Minimal API framework widely adopted in the Node.js ecosystem.
- [FastAPI](https://fastapi.tiangolo.com/) – Python framework for high-performance APIs with automatic OpenAPI docs.

## Databases & ORMs

- [PostgreSQL](https://www.postgresql.org/) – Robust open-source relational database for modern applications.
- [MySQL](https://www.mysql.com/) – Widely used relational database for websites and applications.
- [MongoDB](https://www.mongodb.com/) – NoSQL document database used for flexible data storage.
- [Prisma](https://www.prisma.io/) – TypeScript ORM for relational databases with type-safe queries.
- [Sequelize](https://sequelize.org/) – Promise-based ORM for Node.js with support for SQL databases.
- [SQLAlchemy](https://www.sqlalchemy.org/) – Python ORM for powerful relational data modeling.

## Authentication & Authorization

- [Auth0](https://auth0.com/) – Identity platform for authentication, authorization, and SSO.
- [Firebase Auth](https://firebase.google.com/products/auth) – Authentication with OAuth, email, and phone login.
- [Clerk](https://clerk.dev/) – Authentication platform with prebuilt UI components.
- [Passport.js](https://www.passportjs.org/) – Authentication middleware for Node.js with hundreds of strategies.
- [Devise](https://github.com/heartcombo/devise) – Flexible authentication framework for Ruby on Rails.

## Tooling, Bundlers & Build Systems

- [Vite](https://vitejs.dev/) – Fast development server and build system powered by ES modules.
- [Webpack](https://webpack.js.org/) – Widely used module bundler for modern JavaScript applications.
- [Rollup](https://rollupjs.org/) – Bundler optimized for libraries and tree-shaking.
- [Parcel](https://parceljs.org/) – Zero-config bundler supporting TypeScript, React, and more.
- [ESBuild](https://esbuild.github.io/) – Extremely fast bundler and minifier written in Go.

## Hosting, Deployment & CI/CD

- [Vercel](https://vercel.com/) – Hosting platform optimized for Next.js and modern frontend apps.
- [Netlify](https://netlify.com/) – Jamstack hosting with git-based workflows and serverless functions.
- [DigitalOcean App Platform](https://www.digitalocean.com/products/app-platform/) – Managed hosting for full-stack apps.
- [Render](https://render.com/) – Cloud platform for static sites, APIs, workers, and databases.
- [GitHub Actions](https://github.com/features/actions) – CI/CD automation integrated into GitHub.
- [CircleCI](https://circleci.com/) – Cloud CI/CD solution for automating builds and deployments.

## Testing & QA

- [Jest](https://jestjs.io/) – JavaScript testing framework with built-in mocks and assertions.
- [Mocha](https://mochajs.org/) – Flexible testing framework for Node.js and browser apps.
- [Cypress](https://www.cypress.io/) – End-to-end testing framework for frontend web applications.
- [Playwright](https://playwright.dev/) – Cross-browser testing for UI automation.
- [RSpec](https://github.com/rspec/rspec-rails) – Testing framework for Ruby and Rails applications.

## Web Performance & Optimization

- [Lighthouse](https://developers.google.com/web/tools/lighthouse) – Performance and accessibility auditing tool.
- [WebPageTest](https://www.webpagetest.org/) – Deep analysis of site performance across browsers and locations.
- [PageSpeed Insights](https://pagespeed.web.dev/) – Performance insights powered by real user metrics.
- [ImageOptim](https://imageoptim.com/mac) – Image compression tool for reducing asset size.
- [Bundlephobia](https://bundlephobia.com/) – Estimates package size and performance impact before installing.

## Security

- [OWASP Top Ten](https://owasp.org/www-project-top-ten/) – Essential reference list of top web application security risks.
- [Helmet](https://helmetjs.github.io/) – Middleware for securing Express apps with HTTP headers.
- [Brakeman](https://github.com/presidentbeef/brakeman) – Security scanner for Ruby on Rails.
- [Snyk](https://snyk.io/) – Dependency and container vulnerability scanning.
- [bcrypt](https://github.com/kelektiv/node.bcrypt.js/) – Secure password hashing for Node.js applications.

- [W3Schools](https://www.w3schools.com/) – Beginner-friendly tutorials covering core web technologies.
- [MDN Learning Area](https://developer.mozilla.org/en-US/docs/Learn) – Tutorials provided by Mozilla for modern web development.

### Guides

- [The Odin Project](https://www.theodinproject.com/) – Open-source web development curriculum.
